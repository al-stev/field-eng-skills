**Public function: `generate_dashboard(customer_name, data, output_dir, echarts_path=None)`**

Parameters:
- `customer_name: str` — display name (e.g., "Isomorphic Labs")
- `data: dict` — INTELLIGENCE_DATA dictionary
- `output_dir: Path` — target directory (e.g., `customers/isomorphic-labs/dashboard/`)
- `echarts_path: Path | None` — path to echarts.min.js. If None, look in `SCRIPT_DIR / "lib" / "echarts.min.js"`, then try CDN download, then raise FileNotFoundError with message "echarts.min.js not found. Download from https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js and place at .claude/skills/customer-snapshot/templates/lib/echarts.min.js"

**Steps:**
1. Read `panels.yaml` manifest via `yaml.safe_load()`
2. Determine active panels: panel is active if `always_show: true` OR `resolve_key(data, panel['data_key'])` returns truthy value
3. Read `shell.html` template
4. Replace `{{CUSTOMER_NAME}}` with `customer_name`
5. Replace `{{GENERATED_DATE}}` with `date.today().strftime('%d %b %Y')`
6. Replace `{{PANEL_SCRIPTS}}` with `<script src="panels/{id}.js"></script>` tags for each active panel (sorted by `order` field)
7. Create output directory structure: `output_dir/`, `output_dir/panels/`, `output_dir/lib/`, `output_dir/history/`
8. Write `output_dir/index.html` with processed shell
9. Write `output_dir/data.js` as `const INTELLIGENCE_DATA = {json.dumps(data, default=str, indent=2)};`
10. Copy active panel JS files from `SCRIPT_DIR / "panels" / f"{panel_id}.js"` to `output_dir / "panels" / f"{panel_id}.js"` — skip if source doesn't exist (panel not yet implemented)
11. Copy lib files: `echarts.min.js`, `chart-helpers.js`, `panel-registry.js` to `output_dir / "lib" /`
12. History snapshot: if `output_dir / "data.js"` already existed before step 9, copy old content to `output_dir / "history" / f"data-{date.today().isoformat()}.js"`
13. Compute and return a summary dict: `{ 'output_dir': str, 'panels_active': [list of active panel ids], 'panels_skipped': [list of inactive panel ids], 'data_js_size_kb': int }`

**Helper: `resolve_key(data, dot_path)`**
Walk a dot-separated path (e.g., "usage.support_tickets") through nested dicts. Return the value or None if any key is missing. Handle `.length` suffix by returning `len(value)` if value is a list.
