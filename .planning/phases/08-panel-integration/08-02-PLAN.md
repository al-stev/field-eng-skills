  - ECharts heatmap chart. Height 400px (adjust based on cohort count).
  - xAxis: period labels M+0 through M+11 (or however many periods exist)
  - yAxis: cohort month labels with `n=SIZE` annotation (e.g., "2025-06 (n=12)")

  - Tooltip showing: "Cohort: {month}, Period: M+{n}, Retention: {pct}%"
  - Label show on heatmap cells: `label: { show: true, formatter: function(p) { return p.value[2] !== null ? Math.round(p.value[2]) + '%' : ''; } }`

- **Retention Curve section** (section-label "RETENTION CURVE"): Panel-card in two-col layout (left).
  - ECharts line chart. Height 320px.
  - xAxis: period numbers (0, 1, 2, ...)
  - yAxis: retention percentage (0-100%)
  - series `type: 'line'` from `data.retention_curve`, smooth true, areaStyle with gradient fill.
  - dataZoom for 12+ periods: `[{ type: 'inside' }, { type: 'slider', height: 20, bottom: 0 }]`

- **User Lifecycle section** (section-label "USER LIFECYCLE"): Panel-card in two-col layout (right), conditional on `data.lifecycle` being non-null.
  - ECharts stacked area chart. Height 320px.
  - xAxis: month labels from `data.lifecycle[].month`
  - 4 series (stacked area): New (green), Retained (blue), Resurrected (amber), Churned (red -- negative values plotted below axis)
  - If `data.lifecycle` is null/empty, render partial empty state: "Lifecycle data not available for this account"

- Return `{ charts: [all echarts instances] }`

**getHeadlineStats(data):**
